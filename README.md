# Weather-Forecast-Android-App

Weather Forecast Mobile App for Android implemented via Koltin dedicated packages:
- Gson
- Retrofit
- Kodein
- Room
- Kotlinx Coroutines
- Lifecycle Livedata and ViewModel
- Gms location provider
- ConstraintLayout
- Material Theme
- Glide
